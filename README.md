# UnrealMarkdown
A simple markdown implemented for Unreal Engine.
I made it with UE 5.0.3
![overview](https://github.com/zhuangbiiii/UnrealMarkdown/blob/main/overview.png)

## Markdown Asset Support

- Added a native `Markdown Asset` type for storing raw `.md` content inside Unreal.
- Supports drag-and-drop import of `.md` files into the Content Browser.
- Supports creating blank Markdown assets from the editor.
- Adds `Import Markdown` and `Export Markdown` buttons to the editor toolbar and Tools menu.
